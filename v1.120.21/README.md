U4A 애플리케이션 샘플 - v1.120.21
해당 폴더는 U4A Workspace v1.120.21 환경에서 사용할 수 있는 U4A 애플리케이션 샘플을 제공합니다.

버전 정보
U4A Workspace Version : v1.120.21
폴더 구조 예시
v1.120.21/
 ├─ SampleApp.U4A
 ├─ DemoApp.U4A
 └─ ...
각 .U4A 파일은 U4A Workspace에서 Import 하여 바로 사용할 수 있습니다.

사용 방법
원하는 .U4A 파일을 다운로드합니다.
U4A Workspace를 실행합니다.
상단 메뉴의 Import 기능을 선택합니다.
다운로드한 .U4A 파일을 선택합니다.
Import 완료 후 애플리케이션을 확인합니다.
참고 사항
해당 샘플은 v1.120.21 버전을 기준으로 작성되었습니다.
다른 버전에서는 일부 기능 또는 화면 구성이 다르게 동작할 수 있습니다.
